# Albion Online - Calculateur de Raffinage

Un calculateur de rentabilité pour le raffinage dans Albion Online, utilisant l'API d'Albion Online Data Project.

## Installation

1. Assurez-vous d'avoir Python installé.
2. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
   ```

## Démarrage

Lancez le serveur :
```bash
python app.py
```

Ensuite, ouvrez votre navigateur à l'adresse : `http://localhost:8000`
